Deskripsi:

	Aplikasi Rent-A-Car merupakan aplikasi manajemen sistem bisnis persewaan mobil.
	Aplikasi ini dikembangkan dengan menerapkan pemrograman berorientasi objek dengan 
	menggunakan bahasa pemrograman C# dan framework .NET. Adapun, IDE yang digunakan
	dalam proses pengembangan aplikasi ini adalah Microsoft Visual Studio 2019
	
	Aplikasi ini dapat mengerjakan hal-hal sederhana seperti
	Create, Read, Update, Delete, dan Edit. (CRUDE). Implementasi proses Create 
	dalam aplikasi ini  adalah aplikasi ini dapat  membuat informasi dan memasukan
	informasi kedalam database. Dalam aplikasi ini, proses create terdapat dalam
	kelas Add_Cars dan  Add_Members. Implementasi proses Read dalam aplikasi ini
	adalah aplikasi ini dapat melihat data yang telah dimasukan kedalam database.
	Proses ini terdapat dalam kelas View_Cars dan View_Members. Dalam kelas View_Cars
	dan View_Members juga terdapat implementasi proses Update yaitu dapat mengedit
	dan memperbarui informasi yang  telah kita masukkan kedalam database.
	Proses Delete dan Edit terdapat pada kelas Return_Car dan Rent_Car dimana kedua
	kelas merupakan inti logika bisnis dari bisnis persewaan (rental) mobil.
	Rent_Car dan Return_Car adalah kelas untuk mengedit database sesuai dengan mobil
	yang disewakan dan member yang menyewa mobil tersebut.
	Rent_Car mengedit database dengan mengurangi jumlah mobil yang tersedia,
	sementara Return_Car dapat menambahkan jumlah mobil yang tersedia dalam database.
	
	Aplikasi ini masih mengimplamantasikan fitur-fitur yang sifatnya mendasar
	dan belum terkoneksi dengan fasilitas pendukung seperti Application Programming Interface (API),
	sehingga masih terbuka ruang untuk perbaikan dan peningkatan.
